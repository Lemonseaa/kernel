# Content Business Line

The content/media line is a future application of the same evidence harness.

It should not be implemented before the quant line proves the evidence loop on real data.

## Likely Evidence

- Content plan and generated asset.
- Publishing workflow trace.
- Platform metrics: views, likes, comments, saves, follows.
- Human preference review.
- Baseline/candidate content strategy comparison.

## Boundary

LoopHarness should judge whether content workflow changes improved outcomes. It should not become a full publishing platform.
